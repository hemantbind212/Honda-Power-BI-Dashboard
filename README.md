# Power BI Dashboard – Sales Performance Analysis

## Project Overview
This project presents an interactive **Power BI dashboard** designed to analyze and visualize sales performance across different regions, products, and time periods.  
The dashboard transforms raw data into actionable insights that help businesses monitor performance, identify trends, and support data-driven decision making.

## Objectives
- Analyze overall sales performance
- Identify top-performing products and regions
- Track revenue trends over time
- Provide insights that support strategic business decisions

## Dataset
The dataset used in this project contains information related to:
- Sales transactions
- Product categories
- Regional performance
- Revenue and profit metrics
- Date and time data for trend analysis

## Key Dashboard Insights
The dashboard provides the following insights:

- **Total Sales & Revenue Overview**
- **Monthly Sales Trends**
- **Top Performing Products**
- **Regional Sales Distribution**
- **Customer or Category Performance Analysis**

## Tools & Technologies Used
- **Power BI Desktop**
- Data Modeling
- Data Cleaning
- DAX (Data Analysis Expressions)
- Interactive Data Visualization
